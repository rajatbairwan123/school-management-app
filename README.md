# School Management System

School Management System is a web-based application built using Laravel to manage school operations efficiently. This system helps manage students, teachers, classes, attendance, and administrative tasks.

## 🚀 Features

- Student Management
- Teacher Management
- Class Management
- Attendance Management
- Admin Dashboard
- User Authentication
- Role-based Access Control
- Responsive Design

## 🛠️ Technologies Used

- Laravel
- PHP
- MySQL
- Bootstrap
- jQuery
- HTML/CSS

## 📁 Modules

- Admin Dashboard
- Student Management
- Teacher Management
- Class Management
- Attendance System
- User Roles & Permissions

## ⚙️ Installation

1. Clone the repository
git clone https://github.com/rajatbairwan/school-management-app.git


2. Go to project directory
cd school-management-app


3. Install dependencies
composer install

4. Copy .env file
cp .env.example .env

5. Generate application key
php artisan key:generate

6. Configure database in .env file
7. Run migrations
